# AppCafe ☕

Ứng dụng quản lý quán cà phê được phát triển bằng Java trên Android Studio.

## 📌 Giới thiệu
AppCafe là ứng dụng hỗ trợ quản lý quán cà phê với các chức năng cơ bản như:
- Quản lý sản phẩm
- Quản lý danh mục
- Quản lý đơn hàng
- Giao diện thân thiện, dễ sử dụng

## 🛠 Công nghệ sử dụng
- Java
- Android Studio
- Gradle
- SQLite

## 📂 Cấu trúc project
```bash
AppCafe/
│── app/
│── gradle/
│── build.gradle.kts
│── settings.gradle.kts
│── gradlew
│── gradlew.bat
```

## 🚀 Cách chạy project

### 1. Clone repository
```bash
git clone https://github.com/tramanhhuynh/AppCafe.git
```

### 2. Mở project
Mở project bằng:
- Android Studio

### 3. Chạy ứng dụng
- Chờ Gradle sync
- Nhấn nút Run ▶ để chạy ứng dụng

## 📱 Chức năng chính
- Hiển thị danh sách món
- Thêm / sửa / xóa sản phẩm
- Quản lý đơn hàng
- Lưu dữ liệu bằng SQLite
